# Overlays

Overlay graphic templates and assets for broadcast lower-thirds, logos, and tickers.

## Structure

```
overlays/
└── templates/
    └── default.json    # Default lower-third template
```

## Template Format

Templates are JSON files defining style, layout, and animation parameters. See `templates/default.json` for the schema.

See [docs/ARCHITECTURE.md](../docs/ARCHITECTURE.md) for full details.
